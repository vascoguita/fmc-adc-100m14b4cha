3.1. Timetag seconds register (upper)

HW prefix: timetag_core_seconds_upper
HW address: 0x0
C prefix: SECONDS_UPPER
C offset: 0x0

8 upper bits of seconds counter. Incremented everytime the coarse counter overflows.
